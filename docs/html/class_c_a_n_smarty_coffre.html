Description détaillée

Classe pour gérer la communication CAN pour le coffre.

Cette classe fournit des méthodes pour envoyer des signaux de détection et de désarmement, ainsi que pour démarrer et arrêter le pinging. Elle a pour avantage d'utiliser une queue d'événements pour les messages, ce qui permet de ne pas bloquer le programme principal. Le formattage des messages CAN est fait dans cette classe, de cette manière le protocole applicatif est sur d'être respecté.
